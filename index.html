The Terminal :
A text input and output environment

Different Terms in Terminal :
1)Command Line :Any interface that is used by entering textual commands(gen Window centric).
2)Terminal :This is type of Command Line(gen Mac centric).
3)Console :A command-line interface used to work with your computer.
4)Shell : A program running on terminal/terminal software.

Types of Shell:
i)Bash :A popular shell on Mac OS and Linux.
ii)Z-Shell :Another shell (Mac default).
iii)Window Powershell :A popular shell on window.

Basic Commands :
1)ls = list files (Show my all files)
2)pwd = print working directory (Where am I?)
3)clear = clear screen

Navigation Command :
Inside and Outside Directories/Folder.
cd = change directory
1)cd directoryName :Go to directory
2)cd .. :back directory/back button

Paths :
Absolute and Relative
1)cd Desktop/CSS (relative path)
2)cd /Users/UserName/Desktop/CSS(absolute path)
3) / root directory
4) ~ home directory

Making Directories :
mkdir = make directory/folder.

Flags :
Flags are characters that we pass with commands to modify their behaviour.
Manual Command :
i)man ls : Give information about ls command.
ii)man mkdir : Give information about mkdir command.
With Flags
1)ls -lShow additional information.
2)ls -aShow hidden files.
3)ls -laShow additional information with hidden files.

Touch Command :
Used to create files
Ex1)touch index.html
Ex2)touch style.css

Deleting Files and Folders :
rm :removes files.
rmdir :removes empty folders.
rm -rf:removes any folders.
